# Chroma Pitch

Chroma Pitch is a beautiful pitch detection audio plugin which doesn't really have anything too special about it other than it is in fact going to be very beautiful.


## Methods implemented thus far:

<ul>
    <li>Zero-Crossing;</li>
    <li>YIN;</li>
</ul>   


## Dependencies

<ul>
    <li>JUCE;</li>
    <li>Eigen;</li>
</ul>
